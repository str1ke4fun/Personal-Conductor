## 打包步骤

### Step 1: 编译桌面应用

```bash
cd apps/desktop/src-tauri
cargo build --release
```

产物: `target/release/conductor-desktop.exe`

### Step 2: 编译 CLI (可选)

```bash
cargo build --release -p conductor-cli
```

产物: `target/release/conductor.exe`

### Step 3: 收集资源

```bash
# Live2D Core
cp -r apps/desktop/src-tauri/resources/live2d/core release/resources/live2d/

# Live2D Hiyori Pro runtime
cp -r apps/desktop/src-tauri/resources/live2d/hiyori/hiyori_pro_en/runtime release/resources/live2d/hiyori/hiyori_pro_en/

# 桌宠形象素材
mkdir -p release/resources/avatar
cp apps/desktop/public/avatar/default.mp4 release/resources/avatar/
cp apps/desktop/public/avatar/original/video.mp4 release/resources/avatar/original/
```

### Step 4: 准备状态模板

```bash
mkdir -p release/state-template
# 复制配置模板 (替换 API key 为占位符)
sed 's/"apiKey": "[^"]*"/"apiKey": "YOUR_API_KEY_HERE"/' state/config.json > release/state-template/config.json
# 导出空 SQLite (运行 migration)
cp state/conductor.sqlite release/state-template/conductor.sqlite
```

### Step 5: 收集文档

```bash
mkdir -p release/docs
cp docs/PRD.md release/docs/
cp docs/当前架构说明-用户操作流程-20260526.md release/docs/
cp docs/交接文档-桌宠渲染与主动对话优化-20260527.md release/docs/
cp docs/交接文档-桌宠工具能力与记忆工作区-20260526.md release/docs/
cp docs/文档同步状态-20260527.md release/docs/
cp docs/INDEX.md release/docs/
```

### Step 6: 打包

```bash
# ZIP 打包
cd release
zip -r ../Personal-Conductor-v0.1.0-internal.zip .
```
