# Personal Conductor 桌面端变更记录

## 2024-05-20: 一体化交互界面重构

### 核心架构变更

**从多窗口弹窗 → 桌宠内嵌面板**
- 移除独立的 `task-panel`、`chat`、`settings` 窗口
- 所有功能整合到单一桌宠窗口中
- 双击桌宠即可展开完整面板

### 前端变更

#### 新增文件
- `src/windows/TaskPanelContent.tsx` - 任务面板内容组件（内嵌版）
- `src/windows/ChatPanel.tsx` - 对话面板内容组件
- `src/windows/SettingsPanel.tsx` - 设置面板内容组件

#### 修改文件
- `src/App.tsx` - 简化为直接渲染 PetWindow，移除 hash 路由
- `src/windows/PetWindow.tsx` - 全面重构，增加：
  - 双模式切换（悬浮/展开）
  - 三标签页布局（任务/对话/设置）
  - 全局快捷键注册
  - 失焦自动收起面板
  - 查看新进展后自动重置状态
  - 专注模式倒计时显示
  - 面板高度拖拽调节
  - 托盘菜单导航事件监听

#### 删除文件
- `src/windows/TaskPanel.tsx`（独立窗口版）
- `src/windows/ChatWindow.tsx`（独立窗口版）
- `src/windows/SettingsWindow.tsx`（独立窗口版）

#### 样式变更
- `src/styles/app.css` - 新增大量样式：
  - 面板展开状态样式与阴影
  - 状态指示器动画（工作中/有更新）
  - 新进展气泡提示
  - 标签页导航样式
  - 对话消息气泡样式
  - 设置表单样式
  - 拖拽调节手柄样式

### 状态语义更新

| 旧状态 | 新状态 | 含义 | 视觉 |
|--------|--------|------|------|
| pending_review | working | 系统正在处理任务 | 🟡 黄色脉冲 |
| nodding | update | 有新进展待查看 | 🟣 紫色脉冲 + 气泡 |
| idle | idle | 正常运行 | 🟢 绿色 |
| quiet | quiet | 专注模式 | ⚪ 灰色 |

### 快捷键支持

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+T` | 展开并切换到任务面板 |
| `Ctrl+Shift+C` | 展开并切换到对话面板 |
| `Ctrl+Shift+Q` | 开启 30 分钟专注模式 |
| `ESC` | 收起面板，回到悬浮模式 |

### 交互优化

1. **面板拖拽调节** - 顶部拖拽手柄可调整面板高度
2. **失焦自动收起** - 点击其他窗口时自动收起到悬浮模式
3. **专注模式倒计时** - 鼠标悬停显示剩余时间
4. **托盘菜单增强** - 所有菜单项直接导航到对应标签页
5. **新进展气泡** - 状态为 update 时桌宠底部显示浮动提示

### 后端变更

- `src-tauri/src/tray.rs` - 更新托盘菜单事件处理，发送 `navigate_to` 事件
- `src-tauri/tauri.conf.json` - 移除多余窗口配置，仅保留 pet 窗口

## 2024-05-20: 纯观察模式重构

### 核心理念
系统自动处理所有任务审批，人只负责查看结果，无需任何点击操作

### 前端变更
- 移除任务卡片上的"通过/跳过/拒绝"按钮
- 改为按时间倒序的进展时间线
- 任务状态标签：处理中/已完成/已跳过
- 提案历史显示"自动批准"标识
- 通知横幅从"需要处理"改为"已完成"

### 后端变更
- `notify_decider.rs` - 更新状态判断逻辑
- 移除人工审批相关的通知路径
- `pending` 状态直接映射为系统内部 `working` 状态
